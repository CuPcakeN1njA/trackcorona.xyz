Corona Virus Tracking API

You can find the API at /api

Written by Charlie Hosier

This API allows you to view infection rates and deaths.

The data set begins from the 5th of Febuary (for now)

There are two filters that can be used:

  • date
  • country

The date filter must be used to select the day. The format for the date is:

mm/dd/yy

I appologise for using the American format :(

An example query can be seen bellow

/api?date=02/05/20&country=The United Kingdom

The source of my information comes from the situation reports from the World Health Organisation, you can find them here.
